Dazzle Aura

Dazzle Aura is a responsive website designed for a beauty parlour. The goal of this project was to create a clean and modern UI where users can easily explore services, pricing, and contact details.


Overview

This website is built to represent a real beauty parlour online. It focuses on simplicity, readability, and a user-friendly layout so that anyone visiting the site can quickly understand the services offered.

Features

- Clean and modern user interface
- Fully responsive design (works on mobile, tablet, desktop)
- Services section with clear details
- Pricing display for better user understanding
- Contact section for easy communication


Tech Stack

- React.js
- HTML
- CSS
